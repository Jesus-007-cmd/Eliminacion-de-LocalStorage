Como Eliminar Elementos del LocalStorage
Para eliminar elementos del local store podemos realizarlo de dos maneras, la primera es eliminando todos los elementos que existen y la segunda es eligiendo un elemento ya sea por número de elemento o elemento en si.

Eliminar todos los elementos

Para eliminar todos los elementos solo es necesario escribir dentro de nuestro script LocalStorage.clear();

Eliminar elementos en especifico

Para eliminar elementos tenemos la opción de localStorage.removeItem(keyName). Donde keyName es la cadena que tiene el nombre de la clave eliminar.
